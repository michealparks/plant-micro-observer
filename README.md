# Plant Micro-Observer App

## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) + [Svelte](https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode).

## Getting started

```
npm install
```

#### Developing
```bash
npm start
```

#### Building
```bash
npm run build
npm run preview
```

#### Deploying
```bash
npm run gh-pages
```
